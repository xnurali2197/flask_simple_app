This is an example of dynamic routing in Flask. The URL parameter /user/{{ name }} is captured by the application and used to create a personalized greeting.

How This Works

When you visit /user/YOUR_NAME, Flask:

  1. Receives your name from the URL
  2. Passes it to the view function as a parameter
  3. Renders this template with your personalized name
  4. Displays the customized greeting
@app.route('/user/<name>')
def user(name):
    return render_template('user.html', name=name)

Try It Yourself
